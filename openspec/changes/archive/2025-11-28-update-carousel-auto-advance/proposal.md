## Why
- カルーセル方式は回答を保存していても、「次へ」ボタン操作を毎回挟む必要があり一問一回答式に比べてテンポが悪いというフィードバックがあった。
- 進捗ドットが画面下にあり、親指操作中に視線を大きく移動しないと現在位置が把握しづらい。
- README も最新の挙動（自動遷移やドット配置変更）を説明できていない。

## What Changes
- 回答を選択した瞬間に自動で次の質問へスワイプし、最終問題ではサマリーを提示するオートアドバンスを追加する。
- カルーセルの進捗ドットを 1回答1画面式と同様にヘッダー近くへ移動し、視線移動を最小化する。
- README の「回答フロー」や各形式の説明を最新状態（カルーセル自動遷移＆上部ドット）に更新する。

- Specs: `openspec/specs/aqj-web-questionnaire/spec.md`
- Code: `web/aqj-carousel.html`, `web/aqj.css`, `README.md`
